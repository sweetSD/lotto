## 유의사항

현재 이 프로젝트에는 Firebase APP Id 등 개인적인 정보가 들어있는
- android/key.properties
- android/app/src/main/AndroidManifest.xml
- lib/const.dart

위 3개의 파일은 제외하고 커밋되어 있습니다.

Flutter를 공부하며 제작된 앱이기 때문에 부족한 부분이 있을 수 있지만 참고 용도로 사용해주시면 감사하겠습니다.
